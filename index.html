Keyword Strategy — Why One Keyword Per Group

Air Conditioning vs HVAC vs RefrigerationOn Yellow Pages, most businesses are tagged under ALL THREE categories. Running separate searches returns 80-90% the same businesses. We use ONE keyword ("air conditioning") and dedup handles the rest. Saves money, same coverage.
Electrician vs Electrical ContractorSame businesses, same issue. "Electrician" returns more results on YP than "electrical contractor". One search wins.
When to run multiple keywordsOnly when they represent genuinely different business types. "Roofer" vs "roof plumber" is a good example — different licences, different businesses, genuine new coverage.

On the 30-40% Call Failed Issue

Root causeNumbers that pass format check (04xx, 10 digits) but are disconnected, ported or inactive. Not invented — just stale data from directories.
Fix in this versionStrict format validation on ingest. Source URL saved for every number. Wrong-state area codes (02/03/08) blocked at export.
Optional improvementNumVerify API ($0.005/number) to confirm active before export. Add this if call fail rate stays above 20%.
